About

부산대학교 기계공학부를 졸업한 후 프로그래밍에 흥미가 생겨 컴퓨터 공학과 백엔드 엔지니어링을 공부하며 개발자로 일하고 있습니다.

자세한 이력은 About 페이지에서 확인하실 수 있습니다.

이 사이트

이 페이지는 Rust 표준 라이브러리만으로 작성한 HTTP 서버에서 직접 제공합니다.

《The Rust Programming Language》 마지막 장 예제에서 출발해 라우터·미들웨어 체인·정적 파일 핸들러를 덧붙이고, Docker · Terraform · Ansible · nginx 로 EC2 위에 배포했습니다.

CI/CD 는 GitHub Actions · OIDC · ECR · SSM 으로 구성했습니다.

소스는 GitHub에서 확인할 수 있습니다.
